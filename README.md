# Advertinfo

**India's 360° Advertising & Brand Visibility Agency**

Premium enterprise website built with Next.js.  
**Tagline:** Visibility That Drives Impact

## Live Preview

- Local: `npm run dev` → http://localhost:3000

## Features

- Strict luxury design system (Apple + Porsche + Rolex inspired)
- 100vh cinematic hero with custom-generated night skyline + illuminated billboards
- Sticky glassmorphism navigation (90px)
- Full suite of services with dedicated pages:
  - Outdoor Advertising
  - ATL Advertising
  - BTL Activations
  - Digital Marketing
  - Influencer Marketing
  - Event Management
  - Political Campaign Management
- Smooth animations (Framer Motion)
- Animated counters, infinite marquees, premium cards
- Interactive Leaflet map on Contact
- Consultation booking modal + production-grade forms
- Custom AI-generated high-end visuals (self-contained in `/public/images`)
- Fully responsive with excellent typography and spacing

## Tech Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS 4
- Framer Motion
- React Hook Form + Zod
- Leaflet (interactive map)
- Sonner (toasts)

## Getting Started

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```
